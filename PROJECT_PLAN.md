# Driving Matrix Project Plan

## Project Goal
Build a secure web app that helps coordinate resident transportation scheduling, assignment, and live operational changes.

## Core Parts
- Frontend web app
- Backend API
- Database
- Authentication
- Scheduling engine
- Dispatch / live update system
- Reporting

## Current Folder Structure
- backend/
- frontend/

## Initial Build Priorities
1. Define stack
2. Define data model
3. Create backend starter app
4. Create frontend starter app
5. Connect frontend to backend
6. Add authentication
7. Add first database tables
8. Build scheduling workflow
9. Add dispatch workflow
10. Add security rules
